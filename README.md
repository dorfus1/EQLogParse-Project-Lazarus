# EQLogParser - Project Lazarus Edition

A customized version of EQLogParser built specifically for the Project Lazarus EverQuest server.

This version includes parser improvements and Project Lazarus-specific combat log support that are not available in the standard release.

---

# Features

## Project Lazarus Support

- Full support for Project Lazarus combat logs
- Correct parsing of Lazarus heal messages
- Heal spell name detection
- Spell damage parsing
- Proc detection
- Pet damage support
- Swarm pet ownership detection
- Wizard Epic swarm pet support
- Improved DPS accuracy
- Improved Healing accuracy

---

# Improvements

This version includes numerous Project Lazarus specific parser improvements including:

- Wizard swarm pets correctly linked to their owners
- Improved pet ownership detection
- Better spell parsing
- Better healing spell identification
- More accurate DPS summaries
- More accurate healing summaries
- Improved damage breakdowns
- Improved healing breakdowns

---

# Installation

If you already have EQLogParser installed:

1. Close EQLogParser.
2. Run the included installer.
3. Install to the same folder as your current EQLogParser installation.
4. Allow Windows to replace existing files if prompted.
5. Launch EQLogParser.

---

# Logging

Enable EverQuest logging by typing:

```
/log on
```

EQLogParser reads your EverQuest log files directly.
